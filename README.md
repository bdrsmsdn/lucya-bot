## Getting Started

### This project require NodeJS v14.

### Install
Clone this project

```bash
> git clone https://github.com/Angga23Bot/lucya-bot.git
> cd lucya-bot
```

Install the dependencies:

```bash
> npm install
> npm install -g pm2
```

### How to Install FFMPEG
Just check Tobz Github XD
- [TobyG74](https://github.com/TobyG74/ElainaBOT)

### Usage
1. run the Whatsapp bot

```bash
> pm2 start run.js
> pm2 monit
```
2. stop the Whatsapp bot

```bash
> pm2 stop run.js
```
